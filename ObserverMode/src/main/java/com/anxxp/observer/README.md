 别名： 依赖，发布/订阅（Another Name: Dependents, Publish/Subscribe）

    定义对象间的一种一对多的依赖关系，当一个对象状态发生改变时，所有依赖它的对象都得到通知并
    被自动更新。

    何时使用

    当一个对象的数据更新时，需要通知其他对象，而又不希望和被通知的对象形成紧耦合时
